# guide-cns-hltv

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态网页形式存储，不针对任何特定域名或网站，旨在提供一种简洁、可复用的页面存档与分发方式。

## 项目简介

guide-cns-hltv 是一个纯前端静态页面集合，所有页面均为独立的 HTML 文件，可直接在浏览器中打开。本仓库不依赖后端服务，也不包含动态内容，适合用于存档、展示或作为页面模板参考。

## 目录结构

```
guide-cns-hltv/
├── README.md          # 本文件
├── index.html         # 首页或导航页（如有）
├── pages/             # 存放各类 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 静态资源（CSS、JS、图片等）
    ├── styles/
    ├── scripts/
    └── images/
```

- **pages/**：主要存放独立的 HTML 页面文件，每个文件可独立运行。
- **assets/**：存放页面所需的公共资源，如样式表、脚本和图片。
